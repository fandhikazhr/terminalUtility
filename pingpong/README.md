<h3> Usage </h3>

First compile file `pingpong.cpp`

```
clang++ -Wall -std=c++11 pingpong.cpp -o pingpong
```

## How to play

<h5>Player 1</h5>

- <kbd>w</kbd> for `up`
- <kbd>s</kbd> for `down`

<h5>Player 2</h5>

- <kbd>i</kbd> for `up`
- <kbd>k</kbd> for `down`
